# core-video-to-gif

A JavaScript library to convert video to gif

## Get Started

``` bash
$ npm install core-video-to-gif --save
```

``` js
const coreVideoToGif from 'core-video-to-gif'

coreVideoToGif.convert(videoEle, {
    // options,
    start: 1000, // ms
    end: 1300
}, (err, result) => {
    // handle result
})
```
